QUICK MONGODB SETUP GUIDE
===========================================

If you DON'T have MongoDB installed:

Option A - Install Locally (Windows):
1. Download: https://www.mongodb.com/try/download/community
2. Install with default settings
3. Open PowerShell as Admin:
   net start MongoDB
4. Use: MONGODB_URI=mongodb://127.0.0.1:27017/car_rental

Option B - Use MongoDB Atlas (Cloud - FREE):
1. Go to: https://www.mongodb.com/cloud/atlas
2. Sign up (free)
3. Create Cluster (M0 - FREE tier)
4. Database Access → Add User (username: admin, password: admin123)
5. Network Access → Add IP → Allow from Anywhere (0.0.0.0/0)
6. Clusters → Connect → Connect Application
7. Copy connection string
8. Format: mongodb+srv://admin:admin123@cluster0.xxxxx.mongodb.net/car_rental
9. Paste in MONGODB_URI above


===========================================
AFTER CREATING .ENV FILE
===========================================

1. Save backend/.env file
2. Backend will auto-restart (if running)
3. Check terminal for: "✅ MongoDB Connected"
4. Visit: http://localhost:5000/api/health
5. Create test accounts:
   - Admin: http://localhost:5000/api/create-admin
   - Owner: http://localhost:5000/api/create-owner
6. Start frontend: cd frontend && npm start
7. Open: http://localhost:3000
